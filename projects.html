Gentle Wind

CLI Text Adventure

Timeline: comp. Feb 2025

A text-based adventure game set in a changing, mysterious forest. Terminal based, GENTLE WIND was the first full computer game I created and features dynamicaly changing levels and pathways for the player to navigate through. The random assignment and changing of the paths means that the game has a decent degree of replayability and balances rapid exploration with memory of path and area descriptions.
